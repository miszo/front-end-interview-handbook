Mozilla's frontend loop has historically focused on practical HTML/CSS/JavaScript coding rather than algorithmic problems. The recent process has introduced a CSS-focused online assessment / take-home for some roles.

## User interface coding questions

Mozilla typically asks practical build-up style coding rather than LeetCode. Past loops have included:

- Two "build up" coding questions in HTML, CSS, and JavaScript — features added incrementally on top of an initial implementation.
- CSS take-home / online assessment for certain roles.
